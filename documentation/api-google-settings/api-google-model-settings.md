# API/Google Settings

Configure the default settings for the chatbot to use Google for chat, voice, and image generation. Start by adding your API key then selecting your choices. Don't forget to click "Save Settings" at the very bottom of the page.

More information about Google models and their capability can be found at https://ai.google.dev/gemini-api/docs.
